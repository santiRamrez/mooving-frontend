## Tech Stack

**Requisitos Node:** v18.17.1
**npm version:** 9.6.7

## Ejecuta de manera local

Clone the project

```bash
  git clone https://github.com/santiRamrez/mooving-frontend.git
```

Go to the project directory

```bash
  cd mooving-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
